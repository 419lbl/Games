# Goal = Stop the timer as close to 0.000 as possible
Instructions = Simply open Blender file. Camera is already set to player view, no need to move at all. In text editor, in the top right, play button should be visible. Simply click play button to start game, and hit esc to stop game. Once game has begun, simply hit SPACE to begin timer, and again SPACE when you think timer has reached 0.000. Optional step for moving effects, simply hit SPACE -before- hitting play button to begin game. This will start animations, and set visuals in motion. Now hit play in top right and begin game as usual. To stop effects, simply stop game by hitting esc, push SPACE again to stop effects, then hit play to begin game. Enjoy!
